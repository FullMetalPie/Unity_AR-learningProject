Untitled Unity Project

About the project

"Untitled Unity Project" is a small project developed during my internship.
As i learned how to use unity, i gradually built an app that lets you:
  • Generate 3D image projections form an image in the real world.
  • Roll a ball aroud the detected space, following real life physics.
  • And more...

How i made it

This app was developed using Unity 6 (6000.0.38f1), with the addition of the XRcore and XRfoundation plugins. The entire Unity project is avaiable on GitHub, following this link.
